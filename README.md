# Trabajo-optimizacion-Problema-de-Programacion-entera-0-1
Modelamiento de un problema NP- Completo mediante tres propuestas de lenguaje para comparativas de rendimiento 
Cada carpeta contiene los codigos en su respectivo lenguaje, tanto de su version mejorada aplicando heuristicas y Podas contra la version base del codigo propuesto utilizando Backtracking. Estos rendimientos seran analizados y comparados para determinar las ventajas de un algoritmo sobre otro en temas de rendimiento.
